Provable Data Erasure. Cryptographically Verified.

ZeroTrace is a secure, auditable toolkit for erasing storage devices. It generates tamper-proof certificates to provide definitive proof of data destruction for enterprise, forensic, and privacy-conscious users.

The Erasure & Attestation Workflow

From command to certificate, every step is logged and verified.

Signed, Sealed, and Delivered

The final artifact is more than a log file. It's a portable, verifiable, and tamper-proof JSON or PDF document containing all critical metadata about the erasure event.

  • Device model, serial number, and size
  • Erasure method, passes, and pattern used
  • Cryptographic hashes of pre/post-wipe data
  • Timestamps, duration, and throughput metrics
  • ED25519 signature to detect any tampering
bash 
zerotrace erase \
  --device /dev/sdb \
  --method overwrite \
  --passes 3 \
  --pattern random \
  --sign \
  --cert-out ./certs/sdb_cert.json \
  --yes
